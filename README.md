# Лабораторные работы по курсу  **«Прикладные системы и фреймворки ИИ» (7 семестр, МАИ)**

В этом репозитории собраны лабораторные работы по курсу «Прикладные системы и фреймворки искусственного интеллекта» за 7 семестр МАИ.

Поскольку все лабораторные опираются на общие датасеты и используют схожие метрики качества, общие части (описания данных, метрик и базовую предобработку) вынесены в отдельные файлы.

---

## Структура репозитория

### Общие файлы

- **Metrics description**  
  Описание и объяснение выбранных метрик качества как для задач регрессии, так и для задач классификации. 

- **Datasets description**  
  Описание используемых датасетов: источники, структура, целевые переменные, основные особенности и ограничения, а также краткий анализ качества данных.

- **0) Data preparation**  
  Базовая подготовка данных, общая для всех лабораторных:  
  - очистка и обработка пропусков,  
  - приведение типов,  
  - формирование новых признаков,  
  - первичная дата-аналитика (EDA) и инженерия признаков.  

  Результатом этого шага являются унифицированные таблицы признаков, которые затем используются во всех моделях.

---

### Лабораторные работы и модели

Реализация конкретных моделей и экспериментальные результаты по ним находятся в следующих файлах:

**1.** KNN  
   Реализация моделей на основе алгоритма k-ближайших соседей (K-Nearest Neighbors) для задач регрессии и классификации. 

**2.** Logistic and Linear regression  
   Линейная регрессия для регрессионных задач и логистическая регрессия для задач классификации. Исследование влияния регуляризации, масштабирования признаков и выбора метрик.

**3.** Decision tree  
   Модели на основе решающих деревьев. Анализ глубины дерева, критериев разбиения и влияния переобучения.

**4.** Random forest  
   Случайный лес как ансамбль решающих деревьев. Сравнение с одиночным деревом, исследование влияния числа деревьев, глубины и случайности на качество.

**5.** Gradient boosting  
   Модели градиентного бустинга для задач классификации и регрессии. Анализ влияния шага обучения, числа базовых моделей и сложности базового алгоритма.

---

### Выводы

- **Conclusions**  
  Итоговый файл с выводами по всем лабораторным:  
  - сравнение моделей между собой по основным метрикам,  
  - обсуждение сильных и слабых сторон каждого подхода,  
  - комментарии по улучшению бейзлайнов и влиянию предобработки данных.
